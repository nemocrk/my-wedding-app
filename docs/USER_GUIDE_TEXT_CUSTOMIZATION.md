## 3. Gestione Lingue (i18n)

L'applicazione supporta **Italiano** e **Inglese**.

### Lato Amministratore (Tu)

### Lato Ospite (Invitati)
- Gli ospiti vedranno un selettore di lingua nella loro app.
- **Testi Statici** (pulsanti, menu): Sono tradotti automaticamente dal sistema.
- **Testi Dinamici (Quelli che scrivi tu)**: 
  - Attualmente il sistema supporta l'inserimento di un testo unico. 
  - *Consiglio*: Se hai molti ospiti internazionali, puoi scrivere i testi dinamici in doppia lingua (es. "Benvenuti / Welcome").
